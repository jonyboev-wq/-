# ТопоРедактор (Topoplan Site)

Одностраничный редактор для построения топографического плана по точкам (x,y,z) с генерацией горизонталей. Есть экспорт SVG под формат A2 и масштаб 1:200.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте адрес из консоли (обычно http://localhost:5173).

## Сборка и публикация (например, GitHub Pages, Vercel)

```bash
npm run build
```
Готовые файлы появятся в `dist/`.

## Импорт точек

CSV в формате `x;y;z;id` (разделитель — `;`). Пример есть в интерфейсе.

## Экспорт под A2

Кнопка **«SVG A2 1:200»** формирует векторный файл размером A2 (594×420 мм). Если область не помещается при масштабе 1:200, экспорт автоматически уменьшит масштаб, а фактический масштаб будет вписан в подложку.
