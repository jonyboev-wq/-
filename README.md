# Simple Topographic Editor

Веб‑страница строит топографический план по точкам с высотами и формирует линии рельефа.

## Использование

1. Откройте `index.html` в браузере.
2. Слева указаны исходные точки в формате `x,y,h`. Можно загрузить CSV, изменить значения и нажать «Нарисовать».
3. Настройте шаг сетки и шаг горизонталей. Скрыть или показать сетку и точки можно соответствующими флажками.
4. При построении горизонталей используется интерполяция IDW и библиотека `d3-contour`.
