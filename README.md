# Simple Topographic Editor

Страница позволяет строить простой топографический план с сеткой, точками и линиями рельефа.

## Использование

1. Откройте `index.html` в браузере.
2. Таблица слева уже заполнена исходными координатами точек. Нажмите «Нарисовать», чтобы построить их на плане или измените значения и перерисуйте.
3. Клик по пустому месту добавляет точку без высоты.
4. Клик по одной точке, затем по другой — соединяет их линией.
5. Клик по линии удаляет её.
